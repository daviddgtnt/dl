DavidRP

DavidRP is the resource pack I use every day when I play Minecraft Java Edition. It features the 1.14 textures, CodeCrafted with a connected textures mod for OptiFine, one image that I made with UTK.io that was turned into a resource pack on the Bedrock Edition, a clear UI, a changed progress bar, short swords, and maybe more? It's been a while since I put this together, but I still use it everyday.
